🎯 TeamsBotAgent — Estructura

La clase hereda de AgentApplication del Microsoft 365 Agents SDK. En el constructor registras handlers de eventos. Cada handler es un método async que recibe el contexto del turno.
C# [Agent(name: "AFWebChatBot", description: "Multi-agent bot")] public class TeamsBotAgent : AgentApplication { private readonly AgentRegistry _agentRegistry; private readonly IServiceProvider _serviceProvider; public TeamsBotAgent( AgentApplicationOptions options, IServiceProvider serviceProvider, AgentRegistry agentRegistry, ILogger<TeamsBotAgent> logger ) : base(options) { _serviceProvider = serviceProvider; _agentRegistry = agentRegistry; // Registrar handlers de eventos de Teams OnConversationUpdate( ConversationUpdateEvents.MembersAdded, WelcomeMessageAsync ); OnActivity( ActivityTypes.Message, OnMessageAsync, rank: RouteRank.Last ); } }

💬 OnMessageAsync — Flujo Completo

Cada mensaje del usuario en Teams pasa por aquí:
1. Slash commands → 2. Determinar agente → 3. Crear sesión → 4. Typing indicator → 5. Ejecutar pipeline → 6. Enviar respuesta Markdown.
C# private async Task OnMessageAsync( ITurnContext turnContext, ITurnState turnState, CancellationToken cancellationToken) { var text = turnContext.Activity.Text?.Trim(); // Slash commands (/agents, /agent, /help) if (text.StartsWith('/')) { await HandleCommandAsync(turnContext, text, cancellationToken); return; } // Determinar agente y sesión var agentName = GetAgentForConversation(turnContext) ?? "GeneralAssistant"; var sessionId = $"teams-{turnContext.Activity.Conversation.Id}"; // Indicador de "escribiendo..." await turnContext.SendActivitiesAsync( [new Activity { Type = ActivityTypes.Typing }], cancellationToken); // Ejecutar el MISMO pipeline que usa la web using var scope = _serviceProvider.CreateScope(); var orchestration = scope.ServiceProvider .GetRequiredService<AgentOrchestrationService>(); var request = new ChatRequest(sessionId, text, agentName); var response = new StringBuilder(); await foreach (var evt in orchestration.RunStreamingAsync(request)) { if (evt.EventType == "agent-token") response.Append(evt.Text); } // Teams renderiza Markdown nativamente await turnContext.SendActivityAsync( MessageFactory.Text(response.ToString()), cancellationToken); }

📡 Eventos del AgentApplication

Handlers que registras en el constructor para escuchar diferentes tipos de actividad de Teams.
OnActivity(Message)Mensaje de texto → tu pipeline de agentes
OnConversationUpdate(MembersAdded)Nuevo miembro → welcome message
OnActivity(Invoke)Adaptive Card submit, task modules
OnActivity(MessageReaction)Like, heart, etc. en un mensaje
OnActivity(InstallationUpdate)Bot instalado/desinstalado
C# // Registrar en el constructor: OnConversationUpdate(ConversationUpdateEvents.MembersAdded, WelcomeAsync); OnActivity(ActivityTypes.Message, OnMessageAsync, rank: RouteRank.Last); OnActivity(ActivityTypes.Invoke, OnInvokeAsync); OnActivity(ActivityTypes.MessageReaction, OnReactionAsync);

🃏 Adaptive Cards — Ejemplo

Adaptive Cards ricas con tablas, botones e imágenes. El usuario hace click → el bot recibe la acción vía OnActivity(Invoke).
C# var cardJson = @"{ ""type"": ""AdaptiveCard"", ""version"": ""1.5"", ""body"": [ { ""type"": ""TextBlock"", ""text"": ""Resultado SQL Agent"", ""weight"": ""bolder"", ""size"": ""medium"" }, { ""type"": ""FactSet"", ""facts"": [ { ""title"": ""🥇 Más barato"", ""value"": ""WOM — $12,990"" }, { ""title"": ""🥈 Segundo"", ""value"": ""Entel — $15,490"" } ]} ], ""actions"": [ { ""type"": ""Action.Submit"", ""title"": ""Ver detalle"" }, { ""type"": ""Action.OpenUrl"", ""title"": ""Exportar"", ""url"": ""https://..."" } ] }"; var card = new Attachment { ContentType = "application/vnd.microsoft.card.adaptive", Content = JsonConvert.DeserializeObject(cardJson) }; await turnContext.SendActivityAsync( MessageFactory.Attachment(card), cancellationToken);
